Soldier Shooter

Game Information

Soldier Shooter is a two-dimensional shooter game in which the player runs horizontally and fires at enemies.

How to play:

⚠️ First Download Python ⚠️

Step 1: Download The Game

Step 2: Extract The Zip To Documents Folder

Step 3: Go To The Folder And Click Right Mouse Button

Step 4: Select Open In Terminal

Step 5: Type pip install pygame

Step 6: Close The Terminal And Open Again The Terminal

Step 7: Type python main.py And Click On Enter

Step 8: Now You'e Ready To Play
